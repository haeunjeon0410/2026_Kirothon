## 📌 프로젝트 소개

**쑥맵**은 대학생활에서 흔히 겪는 졸업 설계의 어려움을 해결하기 위해 탄생한 AI 기반 웹 서비스입니다.

> "대학생들은 졸업요건 분석부터 진로 맞춤 수강 설계까지 모두 혼자 해결해야 한다."

학교에서 제공하는 정보가 너무 흩어져 있어, 학생이 정보를 수집하여 졸업 계획을 수립하는 것이 어려운 상황에서 **쑥맵**은 AI가 학점이수표를 분석해 최적의 수강 로드맵을 설계해드립니다.

---

## 😤 Pain Point

대학생의 로드맵 설계는 단순한 과목 조합이 아니라, **졸업과 진로를 동시에 고려해야 하는 복잡한 문제**입니다.

| 문제 영역 | 학생들의 고민 |
|-----------|--------------|
| 🎓 졸업요건 | "이 과목 안 들으면 졸업 못 하는 거 아니야?" |
| 🔗 선수과목 체계 | "운영체제를 안 들으면 네트워크 수강이 막히네…" |
| 💼 진로 적합성 | "개발자가 되려면 어떤 과목부터 들어야 하지?" |
| 📚 전공 유형 | "복수전공·심화전공 조건이 다 달라서 헷갈려…" |
| 🔄 코드쉐어 | "이 과목 다른 전공 학점으로 인정되나?" |
| ✅ 중복 인정 | "이미 들은 과목을 복전 학점으로 인정받을 수 있을까?" |
